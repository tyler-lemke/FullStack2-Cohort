Your DevTools Arsenal

Elements Inspector

Right-click any element and select 'Inspect' to examine its HTML and CSS.

Styles Panel

See all CSS rules applied to an element, including overrides and computed values.

Box Model Diagram

Visualize margins, borders, padding, and content areas of any element.

Copy Selector

Right-click an element in DevTools and select 'Copy selector' to get the exact CSS selector for that element.

Investigation Instructions

  1. Right-click on problem areas and select "Inspect"
  2. Look for crossed-out CSS rules (overridden styles)
  3. Check the computed styles tab for final values
  4. Use the box model diagram to understand spacing
  5. Toggle CSS properties on/off to isolate issues
  6. Modify values directly in DevTools to test fixes
  7. Copy working solutions from DevTools to your CSS file
